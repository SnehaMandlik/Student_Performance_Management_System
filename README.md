🎓 Student Performance Management System — Python

A simple Python console program that takes student names and subject marks as input, then calculates and displays total marks and average score for each student.

📌 What This Program Does

Accepts any number of students as input
Records marks for 3 subjects per student
Calculates total marks and average score
Displays marksheet for all students

🛠️ Tools Used

Language: Python 
Concepts: Loops, Dictionaries, Lists, User Input, String Formatting

▶️ How to Run

bash# Make sure Python 3 is installed
python --version

# Run the program
python Student_Performance_Management_System.py
